# 로건의 아지트 (Logan's Hideout)

## 개요

어린이를 위한 깔끔하고 직관적인 교육용 웹사이트입니다. 학습과 미니게임을 하나의 메뉴에서 즐길 수 있는 싱글 페이지 애플리케이션(SPA) 모델을 사용합니다.

## 스타일 및 디자인

*   **테마:** 가독성을 위한 심플하고 대비가 높은 라이트 모드 테마.
*   **타이포그래피:** 일반적인 한글 산세리프 폰트를 사용하여 가독성을 높입니다.
*   **레이아웃:** 사용자가 모든 게임을 한눈에 볼 수 있는 단일 메뉴를 제공합니다. 게임을 클릭하면 해당 게임 화면으로 전환되며, '홈으로' 버튼으로 메뉴로 돌아올 수 있습니다.
*   **반응형 디자인:** 데스크톱, 태블릿, 모바일 등 모든 화면 크기에 맞춰 레이아웃과 폰트 크기가 자동으로 조절됩니다.
*   **시각 요소:** 동적인 게임을 위해 캔버스를 사용합니다.

## 기능

### 메인 메뉴

*   모든 게임이 통합된 단일 목록을 표시합니다.

### 게임 목록

*   **구구단 게임:** 4~9단의 구구단 문제가 랜덤으로 출제됩니다.
*   **영어 단어 퀴즈:** 초등 수준의 한글 단어를 영어로 번역합니다.
*   **한글 타자기:** 초등 수준의 문장을 따라 치며 타자 속도를 측정합니다.
*   **중력 게임:** 장애물을 피하는 플래피 버드 클론 게임입니다. (PC: 스페이스, 모바일: 터치로 점프)
*   **점핑 게임:** 캐릭터가 악당을 점프로 피하는 플랫포머 게임입니다. (PC: 스페이스, 모바일: 터치로 점프)
*   **슈팅 게임:** 1942 스타일의 비행기 슈팅 게임입니다. 자동으로 총알이 발사되며, PC에서는 방향키로, 모바일에서는 터치 & 드래그로 기체를 조작합니다.

## **프로젝트 리팩토링 계획 (v3.0)**

현재 모든 게임 로직이 `main.js` 파일 하나에 집중되어 있어 코드의 복잡성이 높고, 게임 간의 로직 충돌(예: 중력 게임과 점핑 게임의 충돌)이 지속적으로 발생하고 있습니다. 이는 유지보수와 확장성을 심각하게 저해하는 구조적 문제입니다.

사용자의 제안에 따라, 향후 확장성을 확보하고 코드의 안정성을 높이기 위해 **모듈형 아키텍처**로 전환하는 대규모 리팩토링을 진행합니다.

### 목표

*   **관심사 분리 (Separation of Concerns):** 각 게임의 로직을 독립적인 파일로 분리하여, 하나의 파일을 수정해도 다른 파일에 영향을 주지 않도록 합니다.
*   **확장성 확보:** 새로운 게임을 추가할 때, 기존 코드를 수정할 필요 없이 새로운 게임 파일을 `games` 폴더에 추가하고 `main.js`에 등록하기만 하면 되도록 구조를 개선합니다.
*   **안정성 향상:** 게임 간의 변수 및 상태 공유를 원천적으로 차단하여, 이전에 발생했던 것과 같은 충돌 버그를 근본적으로 해결합니다.

### 실행 계획

1.  **`/games` 디렉토리 생성:**
    *   프로젝트 루트에 `games`라는 새 디렉토리를 만듭니다.
    *   이 디렉토리 안에 각 게임의 로직을 담을 개별 JavaScript 파일을 생성합니다.
        *   `games/multiplication.js`
        *   `games/english.js`
        *   `games/typing.js`
        *   `games/gravity.js`
        *   `games/jumping.js`
        *   `games/shooting.js`

2.  **게임 로직 모듈화:**
    *   각 게임 파일은 자신의 로직을 `export`하는 ES6 모듈 형태로 작성됩니다.
    *   각 게임 모듈은 `init`, `cleanup` (필요 시) 등의 표준화된 인터페이스를 외부에 노출합니다. 캔버스 게임의 경우 `update`, `draw` 함수도 포함될 수 있습니다.

3.  **`main.js` 역할 재정의 (총사령관):**
    *   기존의 거대한 `main.js` 파일의 내용을 대부분 삭제하고, 아래의 역할에 집중하도록 재작성합니다.
    *   **이벤트 위임:** 메인 메뉴의 게임 링크 클릭 이벤트를 처리합니다.
    *   **동적 모듈 로딩:** 사용자가 선택한 게임에 해당하는 JS 모듈 파일을 동적으로 `import()` 합니다.
    *   **게임 인스턴스 관리:** 로드된 게임 모듈의 `init` 함수를 호출하여 게임을 시작하고, `cleanup` 함수를 호출하여 게임을 정리합니다.
    *   **중앙 게임 루프 유지:** 캔버스 게임들을 위한 `requestAnimationFrame` 기반의 중앙 게임 루프는 `main.js`가 계속 관리합니다.

4.  **`index.html` 수정:**
    *   `main.js`가 ES6 모듈 문법 (`import`)을 사용하므로, `<script>` 태그에 `type="module"` 속성을 추가합니다.
    *   `<script type="module" src="main.js?v=3.0"></script>`

이 구조적 변경을 통해 '로건의 아지트'는 더 깨끗하고, 안정적이며, 미래를 향해 확장할 준비가 된 애플리케이션으로 거듭날 것입니다.
