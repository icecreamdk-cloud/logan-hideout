# Logan's Hideout (로건의 아지트)

## Overview

A playful and educational website for children, featuring a "Think Room" for learning games and a "Game Room" for mini-games. The application uses a single-page application (SPA) model where selecting a game hides the main menu and displays the selected game's content.

## Style and Design

*   **Theme:** Dark mode with a high-contrast, kid-friendly color palette.
*   **Typography:** Playful and easy-to-read fonts suitable for children.
*   **Layout:** The main view displays the "Think Room" and "Game Room" sections. Clicking on a game title navigates to that specific game's view. A "Back" button is provided to return to the main menu.
*   **Visuals:** Canvas is used for dynamic games like "Falling Square" and "NO-Guri".

## Features

### Main Menu

*   Displays two main sections: "Think Room" and "Game Room".
*   Each section lists the available games.

### Think Room (놀면서 학습하는)

*   **Multiplication Game:** Random multiplication problems (4-9), pass after 3 consecutive correct answers.
*   **English Word Game:** Translate 4th-grade level Korean words to English. Pass after 3 consecutive correct answers.
*   **Korean Typing Game:** Type a 3rd-grade level Korean sentence and see the typing speed (WPM).

### Game Room (즐거운 미니 게임 한가득)

*   **Falling Square:** A Flappy Bird clone where the player controls a square to avoid obstacles.
*   **NO-Guri:** A platformer where a circle jumps over triangle villains.
*   **+/- Game:** Solve addition/subtraction problems within 5 seconds, pass after 3 consecutive correct answers.
